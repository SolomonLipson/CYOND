# Play Store Listing Draft - Cyond Engineers App

## App Name
Cyond Engineers App

## Short Description (max 80 chars)
Field inspection app for engineers: reports, photos, and quotations.

## Full Description
Cyond Engineers App helps field engineers execute end-to-end inspection workflows quickly and accurately.

Key features:
- Lead-wise inspection workflow and guided checklist tabs
- Rule-Out method capture with day-wise observations
- Defect, elements, measurement, and structural information capture
- Drawing/photo upload with inspection context
- Auto-generation of checklist and quotation PDFs
- Engineer comments and CYOND comments in structured format
- Appointment and status sync with backend services

This app is intended for authorized engineer users only.
